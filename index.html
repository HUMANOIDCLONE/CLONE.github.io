CLONE is a lightweight whole-body teleoperation system that achieves comprehensive robot control using only head and hand poses from a VR headset. It enabeles previously unattainable dexterous loco-manipulation tasks, such as picking up an object from the ground and placing it in a distant bin, facilitating the collection of long-horizon interaction data and establishes a foundation for more capable human-robot interaction in both research and practical applications.

CLONE is a lightweight whole-body teleoperation system that achieves comprehensive robot control using only head and hand poses from a VR headset. It enabeles previously unattainable dexterous loco-manipulation tasks, such as picking up an object from the ground and placing it in a distant bin, facilitating the collection of long-horizon interaction data and establishes a foundation for more capable human-robot interaction in both research and practical applications.

How?

Hand Pose Augmentation -- augment reference motions that cover diverse scenarios.

Closed-Loop Error Correction -- correct stepwise errors to overcome accumulated errors over time.

Unified MoE Framework -- learn a wide range of motions with different patterns under a unified policy.
